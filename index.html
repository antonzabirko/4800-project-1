Elements of Deep RL

Overview

A deep reinforcement learning model is a combination of algorithms which interact with an environment to achieve a control function.

Policy

A policy defines the learning agent's way of behaving at a given time. Roughly speaking, a policy is a mapping from perceived states of the environment to actions to be taken when in those states.

Value Function

In a controlled dynamical system, the value function represents the optimal payoff of the system over the interval [t, T] when started at the time-t state variable x(t)=x.
